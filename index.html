Ad Block Detection

Method 1

Create element in external file

Some websites detect ad blockers by using javascript to create an element from an external file that is commonly blocked such as ads.js.

 Checking

Method 2

Javascript variable in external file

Some websites detect ad blockers by using setting a variable in an external file that is commonly blocked such as adframe.js.

 Checking

Method 3

Banner Ad

Some websites detect ad blockers by checking if the height of a banner ad is over 0. If it's blocked the height will be 0.
